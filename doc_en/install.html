Minify

JUI 라이브러리는 스타일만 별도로 사용할 수 있으며, 하나의 패키징된 파일만 로드하면 됩니다.
참고로 컴포넌트의 애니메이션 효과를 발생시키기 위해서는 Animate.css를 로드해야 하며, 관련해서는 차후에 별도로 구현할 예정입니다. 

<link rel="stylesheet" href="jui.min.css" />
<link rel="stylesheet" href="animate.css" />

JUI 라이브러리의 스크립트는 jQuery 기반이므로 관련 관련 파일을 먼저 로드해야 합니다. 

<script src="jquery-1.8.3.min.js"></script>
<script src="jui.min.js"></script>

Debug

JUI 자바스크립트 라이브러리는 디버깅 용도로 각각 별도로 로드할 수 있으며, base와 core는 필수 로드 파일입니다. 

<script src="base.js"></script>
<script src="core.js"></script>

다음으로는 ui 폴더 안의 파일들을 먼저 로드해야 하고, uix의 파일들은 맨 마지막에 로드합니다. 

<script src="base.js"></script>
<script src="core.js"></script>
<script src="ui/button.js"></script>
<script src="ui/combo.js"></script>
<script src="ui/datepicker.js"></script>
<script src="ui/dropdown.js"></script>
<script src="ui/modal.js"></script>
<script src="ui/notify.js"></script>
<script src="ui/paging.js"></script>
<script src="ui/tooltip.js"></script>
<script src="uix/autocomplete.js"></script>
<script src="uix/tab.js"></script>
<script src="uix/table.js"></script>
<script src="uix/tree.js"></script>
<script src="uix/window.js"></script>
<script src="uix/xtable.js"></script>

필요에 따라 사용하지 않는 UI 파일을 로드하지 않아도 되며, uix 폴더의 UI는 ui 폴더의 UI를 사용하기 때문에 주의하시기 바랍니다.
